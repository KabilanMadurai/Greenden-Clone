# 🌿 Greenden - Responsive Plant Website

Greenden is a responsive multi-page website built using **HTML**, **Tailwind CSS**, and **CSS**. The project showcases a clean, modern, and user-friendly interface for a plant-themed website while demonstrating responsive web design principles and utility-first styling with Tailwind CSS.

## 🚀 Features

* Responsive navigation bar
* Beautiful hero section
* Pure CSS image slider (No JavaScript)
* Featured products section
* Dedicated Products page with responsive product cards
* Contact page with a contact form and business information
* Newsletter subscription section
* Modern hover effects and smooth animations
* Mobile-friendly design using Tailwind CSS responsive utilities

## 🛠️ Technologies Used

* HTML5
* Tailwind CSS
* CSS3

## 📂 Project Structure

```text
Greenden/

```

## 📱 Responsive Design

The website is fully responsive and adapts seamlessly to:

* 📱 Mobile Devices
* 💻 Laptops
* 🖥️ Desktop Screens

using Tailwind CSS responsive classes (`sm:`, `md:`, `lg:`).
